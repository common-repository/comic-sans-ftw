Any Comic Sans lovers out here? This plugin gives you a complete Comic Sans website, right away! Obviously, this plugin is pun intended.

== Description ==

Changes the font of your WordPress frontend and administration panel to Comic Sans. 

== Installation ==

1. Upload `comic-sans-ftw.php` file to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enjoy the Comic Sans power!
